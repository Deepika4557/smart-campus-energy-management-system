# Smart Campus Energy Management System

## Project Purpose
A simple IoT-based system to monitor campus energy usage (electricity, temperature, occupancy). It helps detect abnormal consumption early, improves sustainability, and provides the foundation for dashboards and analytics.

## Setup Instructions
1. Download or clone the repository.
2. Use the 3 branches for development:
   - `initial-setup`
   - `progress-update`
   - `final-version`
